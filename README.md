## 📌 Overview

The **Smart Budget Allocation Tool** is a lightweight, intelligent web application designed to help users:

- Track income and expenses  
- Analyze spending behavior  
- Improve savings habits  
- Plan financial goals  

> 💡 Built for **simplicity, accessibility, and real-world impact**

---

## 🧠 Core Concept

This project uses an **AI-inspired rule-based system** instead of heavy machine learning.

### ✔ What powers it?

- Financial **heuristics** (50/30/20 rule)  
- Rule-based **decision engine**  
- Behavioral **spending insights**  
- Smart **suggestion generator**  

> ⚡ Result: Fast, explainable, and zero-cost intelligence

---

## ✨ Features

### 💰 1. Income & Expense Tracking
- Add monthly income  
- Add categorized expenses  
- Real-time updates  

---

### 📊 2. Budget Analysis
- Total income vs expenses  
- Remaining balance  
- Category-wise breakdown  

---

### 🧠 3. Smart Suggestions (AI-like)
- Detect overspending  
- Suggest cost reductions  
- Improve savings  

**Example:**
> “You are spending too much on food. Try reducing outside meals to save ₹2000/month.”

---

### 📈 4. Financial Health Score
- Score range: **0 – 100**
- Based on:
  - Savings rate  
  - Expense balance  
  - Spending discipline  

---

### 🔮 5. Future Prediction
- Monthly savings projection  
- 1-year estimate  
- 5-year estimate  

---

### 🎯 6. Goal Planner
- Set financial goals  
- Calculate required monthly savings  

---

### 💬 7. Smart Chat Advisor
- Ask questions like:
  - “Can I afford a bike?”  
  - “How can I save more?”  
- Get intelligent responses  

---

### 💾 8. Offline Support
- Uses **localStorage**  
- No login required  
- Works offline
